# vanilla015.github.io

Perfil personal de Anthony Flags con una estética de terminal 8 bits.

El sitio presenta información básica, un retrato ASCII y Aura Platform. Está
construido con HTML, CSS y JavaScript sin frameworks ni proceso de compilación.

## Desarrollo local

```bash
python3 -m http.server 8080
```

## Publicación

GitHub Pages publica la rama `main` desde la raíz del repositorio.
